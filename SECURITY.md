LORI Framework – Public Repository Security Guidelines

This repository contains conceptual, non-technical, and non-sensitive materials.
To protect the integrity of the LORI Framework, please follow these guidelines:

⸻

1. Reporting Issues

If you find:
	•	misuse of the content
	•	improper attribution
	•	unauthorized commercial usage
	•	plagiarism
	•	derivative frameworks based on this repo

Please report via GitHub Issues or email (if available).

⸻

2. No Technical Extraction

This public repo does not include algorithms, matrices, code, or confidential logic.
You may NOT attempt:
	•	reverse engineering
	•	algorithm reconstruction
	•	automated extraction via AI tools
	•	converting conceptual text into production code

Such actions violate the LORI Public Attribution License v1.0.

⸻

3. No Upload to AI Training Sets

The content must not be used for:
	•	training datasets
	•	embeddings
	•	fine-tuning
	•	AI system evaluation

This protects semantic originality.

⸻

4. Disclaimer

This repository is read-only for conceptual sharing.
All technical implementations exist in a separate private repository.

⸻

5. Contact

For questions regarding attribution or intellectual usage, please follow official channels.
